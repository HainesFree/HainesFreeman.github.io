概念

TypeScript是是拥有类型的JavaScript超集，它可以编译成普通、干净、完整的JavaScript代码。

如何理解呢？

  • 紧跟ECMAScript的标准，JavaScript支持的特性，TypeScript都会支持
  • 语言层面增加了语法扩展，如类型约束、枚举类型（Enum）、元祖类型（Tuple）
  • TypeScript最终会编译成JavaScript，所以你不用担心兼容性
  • 更安全、更灵活

特点

  • 始于JavaScript，终于JavaScript
  • TypeScript是一个强大的工具，用于构建大型项目
  • 拥有更先进的JavaScript
    • 拥有最新的和不断发展的JavaScript特性，包含2015年ECMAScript和未来提案中的特性，比如异步功能和Decorators
    • 这些特性为高可信应用程序开发时是可用的

趋势

很多大型项目的源码都是采用TypeScript编写的，如：

  • Vue3
  • Angular
  • VSCode编辑器
  • Ant Design

并且，目前非常流行的开发模式：Vue3 + TypeScript、React + TypeScript。

  • tsc 全称是typescript compiler
  • Babel
