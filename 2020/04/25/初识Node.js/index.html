初识Node.js

Posted on 2020-04-25

目录

概念

Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行时环境。

以下问题可以帮助你理解Node.js概念：

什么是JavaScript运行环境？

  • 客户端-浏览器
  • 服务端-Node

为什么需要JavaScript引擎？

因为我们编写的JavaScript代码不能直接被识别。

  • 编写的JavaScript代码无论运行在浏览器还是Node，最终都是被CPU去执行
  • 但是CPU只认识机器语言，JavaScript代码需要转变为机器语言，才能被CPU执行
  • JavaScript引擎就是帮我们将JavaScript代码翻译成CPU认识的机器语言

什么是V8引擎？

V8引擎属于JavaScript引擎中的一种，常见的JavaScript引擎有以下几种：

  • SpiderMonkey：Brendan Eich开发，也是第一款JavaScript引擎

  • Chakra:微软开发，用于IE浏览器

  • JavaScriptCore:apple公司开发，WebKit中的JavaScript引擎

  • V8：Google开发，用于Chrome和Node.js

浏览器内核包含什么？

主要由两部分组成：

  1. 渲染引擎-处理html、css并将其绘制到页面
  2. JS引擎-解析和执行js来实现网页的动态效果

浏览器内核和V8引擎有什么关系？

V8引擎是浏览器内核组成的一部分

渲染引擎解析html过程中，遇到了js该怎么办？

会停止解析html,而去执行js,那为什么这样呢？因为如果先执行完html的话，js可能会操作Dom改变其html结构，最终导致渲染引擎再去解析html

常见的浏览器内核有哪些？

Trident: 微软开发，IE浏览器

Webkit: 苹果公司开发，用于Safari， 之前Chrome也在用

Blink: 谷歌开发，Webkit的一个分支，Chrome目前在用

安装

  • 官网下载安装傻瓜式操作：https://nodejs.org/en

  • 下载安装成功后，终端运行：node --version，若显示版本号,则代表安装成功！

应用场景

  • 高级前端进阶必经之路
  • 市场招聘要求
  • 作为web服务器，提供服务接口
  • 前端开发的库都是基于Node包管理
  • SSR
  • Node中间层
  • 前端工程化-脚手架
  • 桌面应用-Electron
  • 自动化脚本工具

REPL

概念：

  • Read - Eval - Print - loop 即：读取-求值-输出-循环
  • REPL是一个简单的交互式的编程环境

使用方法：

  • 浏览器中直接f12检查，即可输入js代码调试运行
  • 终端中，输入node后回车，在输入js代码运行
